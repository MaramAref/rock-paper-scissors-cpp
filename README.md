# 🪨 Rock Paper Scissors - C++ 

A simple console-based Rock, Paper, Scissors game built with C++. 

> ⚠️ This is a learning project created to practice programming fundamentals, problem-solving, functions, enums, structs, and modular program design. 

--- 

## 📌 Features 

- Play against the computer.
- Choose between Rock, Paper, or Scissors.
- Play multiple rounds.
- Display round-by-round results.
- Determine the overall winner.
- Color-coded results in the console.
- Play again without restarting the program. 

--- 

## 🛠️ Technologies 

- C++
- Console Application
- Visual Studio

## 📚 Concepts Practiced

- Random Number Generation
- Input Validation
- Game Logic

1. Clone the repository.
2. Open the project in Visual Studio.
3. Build and run the project.
